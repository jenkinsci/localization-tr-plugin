Daha sonra kullanılabilmesi için, yapılandırma artefaktlarını arşivler (mesela, zip veya jar dosyaları gibi) Arşivlenmiş dosyalara, Jenkins web sayfası üzerinden erişilebilir.
Normalde, Jenkins yapılandırma logları tutulduğu müddetçe artefaktları saklar, fakat diskte yer kaplamasından dolayı eski artefaktları istemiyorsanız, silebilirsiniz.
